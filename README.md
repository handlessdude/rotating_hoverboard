# rotating_hoverboard
 <h2>just fun js app with rotating board of glowing squares.</h2>
<h3>controls:</h3>
<ul>
<li>click __left mouse button__ on square or hit __space__ to turn the board</li>
<li>press __escape__ to close the color picker window</li>
</ul>
